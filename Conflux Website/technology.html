What is Tree-Graph?


Unlike the Bitcoin and Ethereum blockchains, which are linear in nature, Conflux’s Tree-Graph structure is divergent. Bitcoin and Ethereum can only confirm one block at any given time. Conflux’s Tree-Graph, by contrast, can confirm multiple blocks concurrently. This parallel processing of blocks and transactions, lowers confirmation times, and substantially increases transaction throughput.
Conflux incorporates all valid blocks, even concurrent ones, into a single data structure called the Tree-Graph. This Tree-Graph ledger, along with the GHAST chain selection mechanism, facilitates the order of execution of all transactions in the system in a scalable and robust manner.
